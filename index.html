overview


abstract

COOL is a prototypical COntent Oriented Lisp derivative whose goal is to be a general-purpose programming language with a different message-passing and I/O programming model.

features

Local and global threaded actors

One-shot threads

Value and environment (closure) serialization

Uniform file and network I/O API based on names

work in progress

Runtime support for lightweight (green) threads

Structure or record types

Immutable field enforcement and type checks

More efficient serialization

Data access control via encryption (i.e., sealed values)

examples

            
                COOL> print "Hello, world!"
                (print 'Hello, world!')
                'Hello, world!'
